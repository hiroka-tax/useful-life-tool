# 耐用年数判定ツール

法人税における減価償却資産の耐用年数を検索できるツールです。

🔗 **[ツールを開く](https://hiroka-tax.github.io/useful-life-tool/)**

---

## 機能

- キーワードを入力するだけで耐用年数を検索
- **e-Gov 法令**（減価償却資産の耐用年数等に関する省令 別表第一〜第十一）と**国税庁 質疑応答・通達**を横断検索
- 関係法令・通達番号を表示
- スマートフォン対応

## 検索例

| キーワード | 対象資産の例 |
|---|---|
| `建物` | 鉄筋コンクリート造・木造など構造別の建物 |
| `LAN` | ハブ・ルーター・光ケーブルなどLAN機器・配線 |
| `車両` | 普通自動車・貨物車・特殊車両など |
| `電気設備` | 照明・分電盤など電気工事 |
| `パーティション` | 可動間仕切り（簡易・スチール製など） |
| `太陽光` | 太陽光・風力発電システム |
| `災害報知` | 消火・排煙・避難設備 |

## データソース

| ソース | 内容 |
|---|---|
| [e-Gov 法令検索](https://laws.e-gov.go.jp/) | 減価償却資産の耐用年数等に関する省令（昭和40年大蔵省令第15号） |
| [国税庁](https://www.nta.go.jp/) | 質疑応答事例・個別通達（LAN配線、太陽光発電など） |
